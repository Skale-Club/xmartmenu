# Plan: Sistema de Pedidos Diretos

## Escopo
Adicionar sistema de pedidos interno no menu público onde clientes escolhem itens, inserem nome/telefone econfirmed o pedido. Pedidos aparecem no store admin e store staff.

## 1. Banco de Dados

### 1.1 Novas tabelas
- `orders`: id, tenant_id, customer_name, customer_phone, status, total, created_at, updated_at
- `order_items`: id, order_id, product_id, quantity, price, notes

### 1.2 Nova coluna em tenant_settings
- `direct_orders_enabled: boolean` (default: false) - ativa/desativa opção de pedido direto

---

## 6. Atualização de Types

### 6.1 Modificar database.ts
Adicionar interfaces:
```typescript
export interface Order {
  id: string
  tenant_id: string
  customer_name: string
  customer_phone: string
  status: 'pending' | 'confirmed' | 'completed' | 'cancelled'

  created_at: string

  price: number
```

## 7. Fluxo do Usuário

### Customer (Menu Público)
1. Escaneia QR code → acessa menu público
2. Visualiza produtos
3. Clica "Add to Cart" nos produtos desejados
4. Clica no carrinho → abre modal
5. Insere nome e telefone
6. Clica "Confirm Order"
7. Pedido criado e aparece no admin/staff

### Store Admin / Staff
1. Acessa `/orders`
2. Visualiza lista de pedidos
3. vê detalhes (itens, cliente)
4. Admin pode mudar status (confirmed/completed/cancelled)
