```tsx
<Image src={url} alt="..." fill priority className="object-cover" sizes="100vw" />
```

For fixed-dimension elements (logo, avatar):
```tsx
<Image
  src={url}
  alt="Restaurant logo"
  width={80}
  height={80}
  className="rounded-xl object-cover bg-white/10 p-1"
  priority
/>
```

Note: `fill` requires parent to have `position: relative|absolute|fixed` and explicit dimensions.
