```typescript
// sanitizeForPrompt strips `{}<>\n\r` and backticks; normalizes whitespace; truncates to maxLength
function sanitizeForPrompt(str: string, maxLength = 100): string {
  return str
    .replace(/[`{}<>\n\r]/g, '')
    .replace(/\s+/g, ' ')
    .trim()
    .slice(0, maxLength)
}
```

Create `.github/workflows/image-seeding.yml` with this exact content:

```yaml
name: Image Seeding

on:
  workflow_dispatch:
    inputs:
      job_id:
        description: 'ai_jobs row ID'
        required: true
      tenant_id:
        description: 'Tenant UUID'
        required: true
      product_id:
        description: 'Single product UUID (empty string = bulk seed all products)'
        required: false
        default: ''

jobs:
  seed-images:
    runs-on: ubuntu-latest
    timeout-minutes: 15
    steps:
      - uses: actions/checkout@v4

      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Run image seeding script
        run: npx tsx scripts/seed-images.ts
        env:
          JOB_ID: ${{ inputs.job_id }}
          TENANT_ID: ${{ inputs.tenant_id }}
          PRODUCT_ID: ${{ inputs.product_id }}
          GOOGLE_GENERATIVE_AI_API_KEY: ${{ secrets.GOOGLE_GENERATIVE_AI_API_KEY }}
          NEXT_PUBLIC_SUPABASE_URL: ${{ secrets.NEXT_PUBLIC_SUPABASE_URL }}
          SUPABASE_SERVICE_ROLE_KEY: ${{ secrets.SUPABASE_SERVICE_ROLE_KEY }}
          VERCEL_REVALIDATE_URL: ${{ secrets.VERCEL_REVALIDATE_URL }}
          VERCEL_REVALIDATE_SECRET: ${{ secrets.VERCEL_REVALIDATE_SECRET }}
```

The script must:

1. Read env vars: `JOB_ID`, `TENANT_ID`, `PRODUCT_ID` (empty = bulk), `GOOGLE_GENERATIVE_AI_API_KEY`, `NEXT_PUBLIC_SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, `VERCEL_REVALIDATE_URL`, `VERCEL_REVALIDATE_SECRET`


3. Update job status to `running` immediately:

4. Fetch tenant data (name, slug, business_type from tenant_settings):
```typescript
const { data: tenant } = await supabase
  .from('tenants')
  .select('id, name, slug, tenant_settings(business_type)')
  .eq('id', tenantId)
  .single()
const businessType = (tenant?.tenant_settings as { business_type?: string } | null)?.business_type ?? ''
```

5. Inline `sanitizeForPrompt` function (copy from src/lib/ai/sanitize.ts — strips `` `{}<>\n\r ``, normalizes whitespace, truncates to 100 chars). Apply to `tenant.name` and `businessType` before any prompt interpolation (per D-14).

6. Initialize GoogleGenAI:
```typescript
import { GoogleGenAI } from '@google/genai'
const ai = new GoogleGenAI({ apiKey: process.env.GOOGLE_GENERATIVE_AI_API_KEY! })
```

7. Helper function `generateAndUploadImage(prompt: string, storagePath: string, aspectRatio: '16:9' | '1:1'): Promise<string>`:
   - Calls `ai.models.generateContent` with `model: 'gemini-3.1-flash-image-preview'`, `responseModalities: ['IMAGE']`, `imageConfig: { aspectRatio }` — NO `imageSize` (silently ignored, Pitfall 4)
   - Extracts base64 from `response.candidates[0].content.parts.find(p => p.inlineData)?.inlineData?.data`
   - Throws if no inlineData found
   - Converts: `const pngBuffer = Buffer.from(base64, 'base64')`
   - Converts: `const webpBuffer = await sharp(pngBuffer).webp({ quality: 85 }).toBuffer()`
   - Uploads: `await supabase.storage.from('tenant-assets').upload(storagePath, webpBuffer, { contentType: 'image/webp', upsert: true })`
   - Returns: `supabase.storage.from('tenant-assets').getPublicUrl(storagePath).data.publicUrl`

8. **Cover photo logic** (when `PRODUCT_ID` is empty string):
   - Check if tenant already has `tenant_settings.banner_url` — skip if exists (D-09 additive)
   - Cover prompt: `Professional food and hospitality photography. Restaurant interior of a ${safeBusinessType || 'restaurant'} named ${safeCompanyName}. Warm, inviting atmosphere. Soft natural lighting, bokeh background. No people, no text, no logos.`
   - Call `generateAndUploadImage(prompt, '${tenantId}/cover.webp', '16:9')`
   - Update tenant_settings: `await supabase.from('tenant_settings').upsert({ tenant_id: tenantId, banner_url: publicUrl }, { onConflict: 'tenant_id' })`
   - Add 1500ms delay after cover before processing products (per RESEARCH.md sequential recommendation)

9. **Product image logic** (both bulk and single-product):
   - If `PRODUCT_ID` is non-empty: fetch only that product
   - If `PRODUCT_ID` is empty: fetch all products where `image_url IS NULL` and `tenant_id = tenantId`
   - For each product (sequential — NOT Promise.all):
     - Fetch category name: `supabase.from('categories').select('name').eq('id', product.category_id).single()`
     - Product prompt: `Professional food photography. A dish called "${safeProductName}" from the "${safeCategoryName}" section of a ${safeBusinessType || 'restaurant'} restaurant. Close-up, top-down or 45-degree angle, clean white plate, soft natural lighting. Food only. No people, no text, no logos, no watermarks.`
     - Call `generateAndUploadImage(prompt, '${tenantId}/products/${product.id}.webp', '1:1')`
     - Update product: `await supabase.from('products').update({ image_url: publicUrl }).eq('id', product.id)`
     - Add 1500ms delay between products (Pitfall 5 — rate limit protection)

10. **ai_usage logging** (non-blocking, same pattern as Phase 9):

```typescript
await supabase.from('ai_usage').upsert({
```
