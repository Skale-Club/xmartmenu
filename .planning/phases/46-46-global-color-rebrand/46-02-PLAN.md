# Plan 46-02: Surgical bg-primary text-zinc-950 Class Replacement + Atomic Commit

## ATOMIC COMMIT CONSTRAINT

This plan executes the final git commit for the entire Phase 46 rebrand. All changes
from 46-01 (CSS vars, hex fallbacks, gradients, admin defaults) and all changes from
this plan (class replacements) must land in ONE commit. This is a hard requirement from
STATE.md — splitting creates broken intermediate state where dark `text-zinc-950` renders
on red `bg-primary` buttons across the entire application.

The atomic commit task is Task 2 of this plan.

---

## Replacement Rules

The replacement is surgical — ONLY replace `text-zinc-950` where it is paired with
`bg-primary` (static or hover/group-hover variant). Global `text-zinc-950` elsewhere in
the codebase is UNTOUCHED.

| Old class pair | New class pair |
|---|---|
| `bg-primary text-zinc-950` | `bg-primary text-primary-foreground` |
| `hover:bg-primary hover:text-zinc-950` | `hover:bg-primary hover:text-primary-foreground` |
| `group-hover:bg-primary group-hover:text-zinc-950` | `group-hover:bg-primary group-hover:text-primary-foreground` |
