# Plan 48-01: CTA Section Refactor in ClientPage.tsx

## Context

`FooterCTABand` currently renders a constrained glass card (`max-w-[1320px] mx-auto`) floating inside a padded section (`px-8`). Requirements CTA-01 through CTA-05 demand the card extends to full viewport width with a restaurant background image visible behind the text, responsive overlay opacity per breakpoint, and zero changes to heading/text/button content or classes.

The `CtaData` interface (lines 33–37) is missing `bg_image_url?: string`. The image file `public/images/cta-bg.jpg` does not yet exist — a placeholder note is included in the acceptance criteria.

**Critical constraint:** Removing `px-8` from the section and removing `max-w-[1320px] mx-auto` from the card are a single atomic edit — splitting them leaves side gaps or collapsed content.
